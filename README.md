# Simple YouTube Downloader

A Python script to download YouTube Videos (High Quality) or Thumbnails.
Works on Android (Termux) and PC (Windows/Linux/Mac).

## Features
- **Smart Device Detection:** Choose between Phone (saves to Gallery) or PC.
- **High Quality:** Merges best video and audio automatically.
- **Shorts Support:** Works perfectly with YouTube Shorts.
- **Thumbnails:** Extracts high-res thumbnails in JPG format.
- **Additionally:** If it's on a phone, the video or preview will be saved in Downloads,
- If on a PC, one of the two will be saved in the folder with the script.
